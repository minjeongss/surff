# Skeleton UI

## Props

| Attribute |  Type  | Default |    Description    |
| :-------: | :----: | :-----: | :---------------: |
|   width   | number |   60    | Set up the width  |
|  height   | number |   60    | Set up the height |

## Example

```tsx
import { Skeleton } from "surff";

function App() {
  return (
    <>
      <Skeleton />
    </>
  );
}

export default App;
```
